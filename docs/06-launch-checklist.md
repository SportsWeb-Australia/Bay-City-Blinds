# Bay City Blinds Launch Checklist

## Design QA

- Homepage looks premium and professional
- Mobile design is strong
- CTA is visible above the fold
- Product/service sections are clear
- Jackson/Tim trust story is included
- Reviews/proof included
- Layout does not feel generic or AI-generated
- Design matches approved direction

## SEO QA

- Unique title tags
- Unique meta descriptions
- Canonical URLs
- Open Graph tags
- Sitemap exists
- Robots.txt exists
- Heading hierarchy checked
- Alt text added
- Internal links checked
- Schema checked
- Service pages indexed where appropriate
- No accidental noindex
- No broken links
- No orphan pages

## Migration QA

- Existing URLs audited
- 301 redirect map completed
- Redirects tested
- No redirect chains
- Custom 404 page created
- 404 page returns true 404 status

## Performance QA

- Images optimised
- Lazy loading used where appropriate
- No unnecessary JavaScript
- Mobile performance checked
- Layout shift checked
- Fonts optimised
- Lighthouse/PageSpeed reviewed

## Forms and Business One QA

- Free quote form works
- Phone links work
- Email links work
- Lead source captured
- Service interest captured
- Suburb captured
- Internal notification works
- Customer confirmation works
- CRM/Business One mapping checked
- Analytics conversion event checked

## Tracking QA

- Google Analytics installed
- Search Console ready
- Form submission tracked
- Phone clicks tracked
- CTA clicks tracked
- Campaign parameters supported

## Browser QA

- Chrome desktop
- Safari desktop
- Edge desktop
- iPhone Safari
- Android Chrome

## Final Approval

- Client content checked
- Contact details checked
- Legal/privacy checked
- Forms tested
- Site deployed
- DNS checked
- SSL checked
- Search Console submitted
