# Bay City Blinds SEO Strategy

## SEO Goal

Build a search-friendly Astro website that helps Bay City Blinds rank for Geelong blinds, curtains, shutters, outdoor blinds, and related local service searches.

## Primary SEO Themes

- blinds Geelong
- curtains Geelong
- shutters Geelong
- outdoor blinds Geelong
- window furnishings Geelong
- free measure and quote Geelong
- mobile showroom Geelong

## Local SEO Principles

Each important service page should include:
- clear service description
- Geelong/local relevance
- benefits
- product options
- process
- reviews/proof
- FAQs
- internal links
- quote CTA

## Technical SEO Requirements

Check or implement:
- title tags
- meta descriptions
- canonical URLs
- Open Graph tags
- sitemap.xml
- robots.txt
- clean URL structure
- heading hierarchy
- image alt text
- structured data
- internal links
- crawlable navigation
- fast loading
- mobile responsiveness
- no broken links
- no important content hidden behind client-only JavaScript

## Schema To Consider

Use schema only where appropriate and supported by visible page content.

Potential schema:
- LocalBusiness
- HomeAndConstructionBusiness
- Service
- FAQPage
- Review snippets only if compliant and visible
- BreadcrumbList

## 301 Redirects

If replacing an existing site, create a redirect map.

Every important old URL should redirect to the closest relevant new URL.

Do not redirect every old page to the homepage unless there is no useful alternative.

Avoid redirect chains.

## 404 Page

Create a useful custom 404 page that:
- returns a true 404 status
- explains the page was not found
- links to major services
- links to free measure and quote
- includes contact options
- helps users recover
